# 家庭记账本 - 项目部署文档

## 环境要求

- Node.js 18 或以上
- npm 9+

## 本地运行

### 1. 后端

```bash
cd backend
npm install
cp .env.example .env   # Windows: copy .env.example .env
npm run seed           # 可选：生成演示账号 demo/123456
npm run dev
```

后端默认地址：`http://localhost:3000`

### 2. 前端

```bash
cd frontend
npm install
npm run dev
```

前端默认地址：`http://localhost:5173`（已配置 API 代理到 3000 端口）

### 3. 生产构建

```bash
# 后端
cd backend && npm start

# 前端
cd frontend && npm run build
# 将 dist 目录部署到 Nginx 等静态服务器，并反向代理 /api 到后端
```

## 目录说明

| 目录 | 说明 |
|------|------|
| backend/ | Express + SQLite API |
| frontend/ | Vue3 + Element Plus + ECharts |
| backend/sql/init.sql | MySQL 建表脚本（课程文档用） |
| backend/data/ | SQLite 数据库文件（运行后自动生成） |

## 常见问题

1. **登录失败**：先执行 `npm run seed` 创建演示账号，或自行注册。
2. **端口占用**：修改 `backend/.env` 中 `PORT` 及 `frontend/vite.config.ts` 代理目标。
3. **跨域**：开发环境由 Vite 代理处理；生产环境同一域名访问页面与 `/api` 即可。
4. **公网打不开**：检查云厂商安全组是否放行 80 端口。
