## 最终部署方案 v2.0

### 新版本功能
- **家庭组系统**：创建/加入家庭组（邀请码+密码），支持多家庭组
- **角色权限**：创建者 / 管理员 / 成员，三级权限管理
- **日历视图首页**：每日收支汇总显示，颜色深度标识
- **记账审批流程**：成员申请记账 → 管理员/创建者审核
- **真实用户关联**：加入家庭组的用户自动同步为成员
- **虚拟"额外"成员**：用于记录宠物等非用户收支

### 部署步骤

1. 购买华为云 ECS（Ubuntu 24.04），安全组放行 80 端口
2. 安装 Node.js 22：
   ```bash
   curl -fsSL https://deb.nodesource.com/setup_22.x | bash -
   apt install -y nodejs nginx
   ```
3. 克隆项目（如已存在则 git pull 更新）：
   ```bash
   git clone https://github.com/qiiq11/FamilyBudgetBook.git
   cd FamilyBudgetBook
   ```
4. 构建前端：
   ```bash
   cd frontend && npm install && npm run build
   ```
5. 安装后端依赖并创建 .env：
   ```bash
   cd ../backend && npm install --omit=dev
   ```
   `.env` 内容：
   ```
   NODE_ENV=production
   PORT=3000
   HOST=0.0.0.0
   JWT_SECRET=<随机字符串至少32位>
   DB_PATH=./data/family_budget.db
   ```
6. 配置 Nginx（80 → 前端静态文件 + SPA fallback，/api/ → 后端 3000 端口）：
   ```bash
   sudo tee /etc/nginx/sites-available/family-budget << 'NGINX_EOF'
   server {
       listen 80;
       server_name _;

       client_max_body_size 10m;

       # 后端 API 反向代理
       location /api/ {
           proxy_pass http://127.0.0.1:3000/api/;
           proxy_http_version 1.1;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
           proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
           proxy_set_header X-Forwarded-Proto $scheme;
       }

       # 前端静态文件 + SPA fallback
       location / {
           root /root/FamilyBudgetBook/frontend/dist;
           index index.html;
           try_files $uri $uri/ /index.html;
       }
   }
   NGINX_EOF

   sudo ln -sf /etc/nginx/sites-available/family-budget /etc/nginx/sites-enabled/
   sudo rm -f /etc/nginx/sites-enabled/default
   sudo nginx -t && sudo systemctl reload nginx
   ```
7. PM2 启动并设置开机自启：
   ```bash
   npm install -g pm2
   cd /root/FamilyBudgetBook
   pm2 start deploy/ecosystem.config.cjs
   pm2 save && pm2 startup
   ```
8. 生成演示数据：
   ```bash
   cd /root/FamilyBudgetBook/backend
   node src/seed.js
   ```

## 常见问题

**Q：访问公网 IP 打不开？**
检查华为云安全组是否放行 80 端口；`pm2 status` 检查后端状态。

**Q：能开首页但登录失败？**
在服务器执行：`cd /root/FamilyBudgetBook/backend && node src/seed.js`

**Q：重启服务器后数据没了？**
PM2 开机自启 + DB_PATH 指向固定目录，数据持久化。

**Q：升级到 v2.0 后旧数据库不兼容？**
v2.0 数据库结构有变更，需要删除旧数据库重新生成：
```bash
cd /root/FamilyBudgetBook/backend
pm2 stop family-budget
rm -f data/family_budget.db
node src/seed.js
pm2 start family-budget
```
